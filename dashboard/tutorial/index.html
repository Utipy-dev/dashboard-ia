Dashboard IA — Tutorial

Guía de uso para la Etapa 1: solo ordenador, sin servidor.

Qué hace este dashboard

Cuando los tokens de una IA se agotan o la sesión se corta, puedes seguir acumulando ideas, mensajes y temas aquí. Cuando la sesión se renueve, copias y pegas con un clic.

Instalación requerida

Chrome o Edge. Nada más. El dashboard funciona directamente desde el navegador sin ningún servidor.

Cómo abrir el dashboard

  1. Abre dashboard/index.html con Chrome o Edge.
  2. Haz clic en Seleccionar carpeta de datos.
  3. Navega hasta la carpeta dashboard/data/ y selecciónala.
  4. El navegador te pedirá permiso de lectura y escritura — acepta.
Cada vez que abras el dashboard tendrás que seleccionar la carpeta de datos. Es una limitación de seguridad del navegador — no hay forma de evitarla sin instalar un servidor (Etapa 2).

Estructura de datos

Todo se guarda en los archivos JSON de la carpeta data/. Son archivos de texto — puedes abrirlos y editarlos manualmente si hace falta.

  • ias.json — IAs configuradas (Claude, ChatGPT, etc.)
  • proyectos.json — Proyectos por IA
  • conversaciones.json — Conversaciones y temas
  • transcripciones.json — Texto acumulado

Usar con OpenWhispr (dictado)

OpenWhispr funciona a nivel de sistema operativo: el texto cae donde esté el cursor. Para dictar en el dashboard:

  1. Abre la conversación (clic en el icono ⤢ para pantalla completa).
  2. Pon el cursor en el área de texto de transcripción.
  3. Dicta con OpenWhispr normalmente.
  4. Haz clic en Guardar para registrar el fragmento con timestamp.

El dashboard no controla OpenWhispr — solo organiza el resultado.

Cronómetro de renovación

Cada IA tiene un cronómetro en la pestaña superior. Se inicia cuando añades la IA y cuenta hacia atrás según el ciclo que configures. Cambia de color cuando queda poco tiempo.

Es manual — no hay integración automática con las IAs.

Flujo habitual

  1. La sesión de la IA se corta o los tokens se agotan.
  2. Abres el dashboard y sigues acumulando ideas en la conversación pendiente.
  3. Cuando el cronómetro llega a cero (o antes), vuelves a la IA.
  4. Abres la conversación a pantalla completa y haces clic en Copiar todo (limpio).
  5. Pegas en la IA — el texto sale sin metadatos ni timestamps.

Limitaciones de esta etapa

  • Solo funciona en ordenador (Chrome o Edge).
  • Hay que seleccionar la carpeta de datos cada vez que se abre el dashboard.
  • No hay sincronización con el móvil — eso es la Etapa 3.
